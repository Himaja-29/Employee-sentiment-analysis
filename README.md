# Employee Sentiment Analysis

## Overview
This project analyzes employee emails to evaluate sentiment, engagement, and flight risk.

## Steps
1. Sentiment labeling using NLP
2. Exploratory Data Analysis
3. Monthly sentiment scoring
4. Employee ranking
5. Flight risk identification (rolling 30 days)
6. Linear regression modeling

## Tools
- Python
- pandas, sklearn, TextBlob, matplotlib, seaborn

## Key Insights
- Identified top positive and negative employees
- Flagged employees at flight risk
- Modeled sentiment trends using regression
